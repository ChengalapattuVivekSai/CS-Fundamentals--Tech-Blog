# Module 0 - COA to OS Bridge

## Purpose
Connect hardware behavior to operating system mechanisms before entering core OS abstractions.

## Outcome
Explain how privilege, interrupts, address translation, and context state make an operating system possible.

## Linux Focus
- CPU privilege levels
- interrupt path
- syscall trap
- virtual addressing
