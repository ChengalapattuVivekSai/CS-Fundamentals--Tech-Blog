# Module 3 - Processes

## Purpose
Study execution as the operating system sees it: isolated processes with state, memory, and lifecycle.

## Outcome
Explain how programs become processes, how process state changes, and how Linux manages them.

## Linux Focus
- fork
- exec
- wait
- /proc
- process state
