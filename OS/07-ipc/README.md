# Module 7 - Inter-Process Communication

## Purpose
Understand how isolated processes coordinate, exchange data, and trade performance for safety and flexibility.

## Outcome
Choose the right IPC mechanism based on copying cost, synchronization, and system architecture needs.

## Linux Focus
- pipes
- shared memory
- signals
- Unix sockets
- file descriptors
