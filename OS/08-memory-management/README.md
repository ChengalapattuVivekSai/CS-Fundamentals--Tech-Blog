# Module 8 - Memory Management

## Purpose
Study how the OS creates the illusion of large, safe, and efficient memory for every running process.

## Outcome
Explain virtual memory, page faults, replacement policies, and Linux memory behavior with precision.

## Linux Focus
- paging
- mmap
- /proc/<pid>/maps
- RSS vs VSZ
- OOM behavior
