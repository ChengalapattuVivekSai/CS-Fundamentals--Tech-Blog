# Module 9 - File Systems and Storage Internals

## Purpose
Understand how the OS stores persistent data, resolves paths, preserves consistency, and exposes files as a stable abstraction.

## Outcome
Explain inodes, path resolution, journaling, and the Linux filesystem hierarchy.

## Linux Focus
- ext4 intuition
- inodes
- VFS
- /proc
- /sys
- /dev
