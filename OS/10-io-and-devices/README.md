# Module 10 - I/O and Device Interaction

## Purpose
Study how the OS moves data between hardware and software while controlling latency, throughput, and concurrency.

## Outcome
Reason about device interaction, blocking vs non-blocking I/O, and multiplexing primitives.

## Linux Focus
- device drivers
- DMA
- file descriptors
- select/poll/epoll
